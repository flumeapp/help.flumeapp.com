# Preferences

To access Flume's Preferences, use the macOS standard <kbd>⌘,</kbd> keyboard shortcut, or access it from the Menu Bar under <kbd>Flume > Preferences…</kbd>.

<p style="text-align: center; margin-top: 1em;"><img src="/preferences/assets/preferences-macos.png" width="60%" height="60%" /></p>

You can also access Flume's Preferences through an additional shortcut under your [Profile's](/views/profile/README.md) Settings menu.

<p style="text-align: center; margin-top: 1em;"><img src="/preferences/assets/preferences-profile.png" width="60%" height="60%" /></p>

> **[info]**
> If you have set Flume to [display its icon in the Menu Bar only](//preferences/general.md#show-icon-in), you will not see Flume's Menu Bar menu at all (by macOS design). To access preferences, right-click the Flume icon in your Menu Bar and choose the <kbd>Preferences…</kbd> menu item.
